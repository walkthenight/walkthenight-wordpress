Mijireh brings Tickera support for over 85 payment gateways in more than 50 countries with a fully customizable, PCI compliant hosted checkout solution.


== Description ==

Use any payment gateway you want, they are all built in for free. Plus, you can change gateways at any time! No matter where you live, you can probably find a gateway that works for you.
